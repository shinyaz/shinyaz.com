`agentcore deploy` の途中でこのエラーが出て一瞬焦った。

```text title="Output"
⚠️ Failed to enable observability: ValidationException: X-Ray Delivery Destination is supported with CloudWatch Logs as a Trace Segment Destination. Please enable the CloudWatch Logs destination for your traces using the UpdateTraceSegmentDestination API
```

結論から言うと無視して問題ない。これはメモリリソース（STM）の作成フェーズでObservabilityを有効化しようとした際に、X-Rayのトレースセグメント先がまだCloudWatch Logsに設定されていないために発生する。

後続のランタイム作成フェーズで `Configured X-Ray trace segment destination to CloudWatch Logs` と出力され、トレースセグメント先が自動設定される。ランタイム本体のObservabilityは正常に有効化されるので、エージェントの動作にもモニタリングにも影響はなかった。
