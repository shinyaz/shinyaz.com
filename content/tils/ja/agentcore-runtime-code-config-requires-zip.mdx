AgentCore Runtimeをコード構成（`codeConfiguration`）で作ろうとして、S3に`main.py`を直接置いたらこのエラーが出た。

```text title="Output"
An error occurred (ValidationException) when calling the CreateAgentRuntime operation: S3 object must be in .zip format. ZIP your code and try again.
```

`zip agent.zip main.py`してからS3にアップロードしたら通った。コンテナ構成ではなくコード構成を使う場合でも、S3上のオブジェクトはZIPでなければならない。
