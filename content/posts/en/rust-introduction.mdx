## Ownership

In Rust, each value has exactly one owner.

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // Ownership of s1 moves to s2
    // println!("{}", s1); // Compile error
    println!("{}", s2);
}
```
