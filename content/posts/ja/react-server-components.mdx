## Server Components vs Client Components

Server Componentsはサーバー側でのみレンダリングされ、バンドルサイズに影響しません。

```tsx
// Server Component (default)
export default async function Page() {
  const data = await fetchData();
  return <div>{data.title}</div>;
}
```
