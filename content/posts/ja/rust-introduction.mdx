## 所有権

Rustでは各値にはただ一つの所有者が存在します。

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1の所有権がs2に移動
    // println!("{}", s1); // コンパイルエラー
    println!("{}", s2);
}
```

## 借用

所有権を移動せずに参照を渡すことができます。
