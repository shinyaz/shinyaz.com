## 言語

- **Python** -- ML/AI、データパイプライン、バックエンドサービスのメイン言語
- **TypeScript** -- Web アプリケーション、API、AWS CDK による IaC
- **Rust** -- システムプログラミングやパフォーマンスが重要なコンポーネント

## AI / ML

- **PyTorch** -- ディープラーニングのモデル学習・実験
- **Hugging Face Transformers** -- NLP タスク、ファインチューニング、推論
- **LangChain** -- LLM アプリケーションのオーケストレーションと RAG パイプライン
- **Amazon Bedrock** -- 本番ワークロード向けマネージド基盤モデル
- **Strands Agents** -- モデル駆動のツールオーケストレーションによる AI エージェント構築 SDK

## クラウド (AWS)

- **Amazon SageMaker** -- モデルの学習・ホスティング・MLOps
- **Amazon ECS** -- マイクロサービスのコンテナオーケストレーション
- **AWS Lambda** -- イベント駆動ワークロード向けサーバーレスコンピューティング
- **AWS Step Functions** -- ML パイプラインのワークフローオーケストレーション
- **AWS CDK** -- TypeScript による Infrastructure as Code

## フロントエンド

- **Next.js** -- サーバーレンダリング・静的サイト向け React フレームワーク
- **React** -- UI コンポーネントライブラリ
- **Tailwind CSS** -- ユーティリティファースト CSS フレームワーク

## データ

- **PostgreSQL** -- メインのリレーショナルデータベース
- **DynamoDB** -- 高スループット・低レイテンシの NoSQL
- **OpenSearch** -- 全文検索とログ分析
- **Redis** -- キャッシュとセッション管理

## エディタ / 開発環境

- **Visual Studio Code** -- メインのコードエディタ
- **Claude Code** -- ターミナルベースの AI コーディングアシスタント
- **Kiro** -- AI エージェント搭載の IDE（spec-driven development）

## DevOps

- **GitHub Actions** -- CI/CD パイプライン
- **Docker** -- 開発・デプロイのコンテナ化
- **Terraform** -- マルチクラウドのインフラプロビジョニング
